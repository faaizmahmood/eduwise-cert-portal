# EduWise – Certificate Verification (Frontend)

A lightweight app to verify certificates issued by EduWise using a unique certificate ID.

## ✅ Why I Built This
This was part of my **Software Engineering term project** and also included in my **first large full-stack system after my internship**.  
I built a separate verification tool so certificates feel authentic and professional — just like real e-learning platforms.

## 🛠 Tech Stack
- **Frontend:** React.js + Bootstrap
- **Backend:** Node/Express
- **Deployment:** Vercel

## ✨ Features
- ✅ Enter certificate ID  
- ✅ Validate against backend  
- ✅ Show details if valid  
- ✅ Show an invalid message if not valid  
- ✅ Clean, simple UI
